1px像素问题

发表于 2020-07-24 更新于 2020-08-08 分类于 css 阅读次数：

开发中总会遇到这样的问题，设计师走查说你这个线好粗啊，可是我明明代码里写的是1px，这是怎么回事呢？其实这就是移动端开发中常见的1px像素问题。

原因

因为css中的1px并不等于设备的1px，这是因为不同的设备，像素比是不一样的。我们可以使用window.devicePixelRatio来获取像素比，计算公式为：像素比 = 设备物理像素/css逻辑像素
通常iphone中的视网膜屏，像素比为2或3，所以当你在css中写1px时，实际上设备上的像素已经是2或3了，因此边框看起来变粗了。
下面我们就来说一说怎么解决这个问题

0.5px边框

0.5px在ios8及以上是支持的，但是在ios7及以下，和安卓设备上，会把0.5px解释称0px，造成边框消失了
